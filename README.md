# buffchat
一个简单的聊天室
server.php 是swoole的进程脚本
index.php 是公共聊天室页面
groupWen.php 是文人雅士聊天室页面
groupFei.php 是死肥宅聊天室页面
public 目录下的 1.js是前端逻辑脚本
class 目录下的WebS类是主类.
process 下的脚本是登录验证脚本
验证码类在public/images 目录下
其他的基本没什么了,
